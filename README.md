Etape 1 : Décrit brièvement le projet
Le projet est un site web qui permet d’ajouter des produits avec leurs caractéristiques/données à une base de données.
Barres de recherches :
	- Product Name : Ajouter le nom du produit
	- Product Price : une case que pour ajouter des chiffres (données décimales)
	- Product Category : Une sélection d’une liste (Phone, TV, Laptob)
	- Product Condition : Une sélection d’une liste (Good, Bad, Excellent)
